Perlbrew Release 0.63

perlbrew 0.63 is released on 2013-05-06T16:53:48+0200 with a few changes

  • Remove install-ack command.
  • Due to the new architecture of ack2, it does not make sense to install the standalone executable anymore. Thus it's better not to do the installation with perlbrew.
  • Fix the use and switch command when switching between binary incompatible perl installations. See GH #297
